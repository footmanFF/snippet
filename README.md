基于主题 [snippet](https://github.com/shenliyang/hexo-theme-snippet) 进行了如下修改：

> 1. 替换公告栏为作者信息栏
> 2. 增加 AddThis 分享模块支持
> 3. 增加畅言社会化评论模块支持
> 4. 优化了正文的排版显示
> 5. 标题以单行显示，超出部分以省略号代替
> 6. 替换首页轮播栏为通知公告栏

配置使用方式参考[原主题说明](https://github.com/shenliyang/hexo-theme-snippet)，在此对原作者的工作表示致谢~
